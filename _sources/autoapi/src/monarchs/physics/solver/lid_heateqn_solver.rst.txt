src.monarchs.physics.solver.lid_heateqn_solver
==============================================

.. py:function:: src.monarchs.physics.solver.lid_heateqn_solver(x, args)

   scipy.optimize.fsolve version of the lid heat equation solver. Solves physics.heateqn_lid.
   Called in lid_functions.lid_development.

   :Parameters: * **x** (*array_like, float, dimension(cell.vert_grid_lid)*) -- Initial guess at the lid column temperature. [K]
                * **args** (*array_like*) -- Array of arguments to pass into the function. See args_array for details.

   :returns: * **root** (*array_like, float, dimension(cell.vert_grid_lid)*) -- Vector containing the calculated lid column temperature, either after successful completion or
               at the end of the final iteration for an unsuccessful solution. [K]
             * **infodict** (*dict*) -- A dictionary of optional outputs, such as the number of Jacobian iterations, etc.
               See scipy.optimize.fsolve documentation for more details on the content of infodict.
             * **ier** (*int*) -- An integer flag. Set to 1 if a solution was found, otherwise refer to mesg for more information.
             * **mesg** (*str*) -- If no solution is found, mesg details the cause of failure.
