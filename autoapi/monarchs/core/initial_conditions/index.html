monarchs.core.initial_conditions

Functions used by run_MONARCHS.py to convert a model runscript (default model_setup.py) to the format actually used by MONARCHS. This includes setting up the initial firn profile information, loading in meteorological data and interpolating it, and loading in/interpolating the digital elevation map (DEM) if applicable.
