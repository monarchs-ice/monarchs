src.monarchs.physics.solver

This module contains functions used to solve the various heat/surface energy balance equations, using the Scipy.optimize.fsolve implementation. This is rather unwieldy, but allows us to select a solver conditionally, and do so both in the main model code using the relevant flag in model_setup.py, but also in our test suite. This is in part a tradeoff between usability of the model, and code clarity. This approach was chosen to maximise usability, so that the different solvers can be generated according to the value of a single Boolean.
